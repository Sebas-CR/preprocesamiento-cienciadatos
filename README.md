# preprocesamiento-cienciadatos

## Objetivo
Implementar un repositorio para la gestión de versiones de código Python enfocado en tareas de preprocesamiento de datos, utilizando Git y GitHub como herramientas principales.

## Estructura Básica de Carpetas
El proyecto sigue una estructura modular simple:

- **src/**: Contiene el código fuente principal del proyecto, incluyendo el archivo `preprocesamiento.py` con las funciones clave
- **data/**: Directorio destinado a almacenar los datasets originales o de prueba.
- **docs/**: Contiene la documentación del proyecto, incluyendo el `DOCUMENTACION.md` y la evidencia de la tarea
